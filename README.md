# Pomodoro Web App
üçÖ Pomodoro timer web app built with React and UI design inspired by Frontend Mentor's [challenge](https://www.frontendmentor.io/challenges/pomodoro-app-KBFnycJ6G).

## Features
* Full control (start/stop timer, change mode)
* User-customizable (Timer intervals, font size, color theme...)
* Alarm sound

## Running
Clone this repo:
```sh
git clone https://github.com/quangthan87/react-pomodoro.git
```

Install dependencies:
```sh
npm install
```

Start the development server:
```sh
npm start
```
